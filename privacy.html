Privacy Policy for DocSenseAI

Last Updated: June 2026

Overview

DocSenseAI is a Chrome extension that helps developers understand technical documentation using AI-powered explanations and contextual search.

Information We Collect

DocSenseAI does not collect personal information such as names, email addresses, phone numbers, financial information, health information, or location data.

The extension may access and process the content of documentation pages that users choose to analyze.

API Key Storage

Users provide their own Google Gemini API key. The API key is stored locally in the user's browser using Chrome storage APIs and is not transmitted to the developer.

Third-Party Services

When users request AI explanations or search results, relevant documentation content may be sent directly to Google's Gemini API to generate responses.

Data Sharing

DocSenseAI does not sell user data and does not use information for advertising or marketing purposes.

Contact

For questions regarding this Privacy Policy, please contact the project maintainer through the GitHub repository.
